# 项目简介

## 核心功能
- 数据采集：使用爬虫技术从豆瓣电影网站抓取电影基本信息、评分、评论等数据。
- 数据清洗与预处理：进行缺失值处理、异常值处理，并执行数据标准化处理。
- 数据分析：分析电影评分分布、电影类型数量分布、演员表现及影响力等。
- 数据可视化：利用Flask框架、Echarts图表库、Python数据分析库（如Pandas、NumPy）和可视化库（如Matplotlib、Seaborn）进行可视化展示。

## 适用场景
- 电影行业分析
- 数据研究人员
- 电影爱好者
- 需要了解电影数据特征与趋势的用户

## 技术特点
- 基于Python的强大数据分析能力
- 使用Flask框架搭建Web应用
- 利用pyecharts、pandas、numpy、matplotlib等工具实现数据可视化
- 交互式界面方便用户查询与探索电影信息
